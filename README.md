📊 Customer Churn Analysis

A data analysis project using the Telco Customer Churn dataset, including visual figures, insights, and business conclusions.

📌 Project Overview

This project focuses on Customer Churn Analysis using the Telco Customer Churn dataset. The goal is to identify patterns and factors that influence customer churn and help businesses make data-driven decisions to improve customer retention.

Customer churn refers to customers who stop using a company’s services. Understanding churn behavior helps organizations reduce revenue loss and increase customer satisfaction.

📂 Dataset Information

Dataset Name: Telco Customer Churn

File Name: WA_Fn-UseC_-Telco-Customer-Churn.csv

Source: IBM Sample Dataset

Total Records: 7,043

Total Features: 21

🔑 Key Columns

customerID – Unique customer identifier

gender – Male / Female

SeniorCitizen – Whether the customer is a senior citizen

tenure – Number of months the customer stayed

Contract – Month-to-month / One year / Two year

MonthlyCharges – Monthly billing amount

TotalCharges – Total billing amount

Churn – Whether the customer churned (Yes / No)

🎯 Objectives

Analyze customer behavior and churn patterns

Identify key factors contributing to churn

Build insights to support customer retention strategies

Visualize churn trends using dashboards

📊 Analysis Steps

Data Cleaning and Preprocessing

Exploratory Data Analysis (EDA)

Churn Distribution Analysis

Feature Impact Analysis

Visualization and Dashboard Creation

✅ Conclusion

The analysis reveals that customer churn is largely driven by pricing, contract type, and customer tenure. Businesses can reduce churn by improving customer engagement, offering long-term contracts, and targeting high-risk customers with personalized retention strategies.
